# Маникюр — сайт записи

Сайт на Flask с двумя сторонами:

1. **Для клиента** — главная страница с фото работ, телефоном и часами работы, кнопка «Записаться онлайн» → страница выбора свободной даты/времени.
2. **Для мастера** — закрытый кабинет (кнопка «Кабинет мастера» в углу главной) с паролем, где можно:
   - добавлять окна для записи (диапазоном или по одному);
   - видеть список записей клиентов (имя, телефон, комментарий);
   - освобождать занятое окно, если клиент отменил запись;
   - загружать/удалять фото для главной страницы.

## Запуск

```bash
cd manicure_site
pip install -r requirements.txt
python app.py
```

Откройте http://localhost:5000

## Пароль мастера

По умолчанию: `master2024`
Поменять можно в `app.py` — переменная `MASTER_PASSWORD`.

## Контакты салона

Телефон, часы работы и адрес — переменная `SALON_INFO` в `app.py`.

## Фон главной страницы

Файл `static/img/bg.jpg` — замените на своё фото, оно автоматически будет размыто через CSS (можно поменять степень размытия в `static/css/style.css`, свойство `filter: blur(7px)` у класса `.hero-bg`).

## База данных

Используется SQLite (`salon.db`), создаётся автоматически при первом запуске.
